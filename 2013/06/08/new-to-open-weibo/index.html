微博开放平台之一入门

微博的开放平台很早就听说了，然后也有一些接触，但只是接触而已，每次想用的时候却都不知道该做些啥。。。前天，在一位女生的威逼利诱下，我才真正开始使用微博开放平台提供的相关API。

Python SDK

微博提供了现有大部分主流编程语言的SDK，当然有些是第三方的，譬如Python的SDK就是一个叫@廖雪峰的人弄出来的。

根据SDK主页上提供的文档，我在SAE上面部署了它的Demo。

我从2011年就开始接触SAE了，曾经在SAE上部署过一个Wordpress博客，但是由于那时需要支付租金，5、6个应用每天的消耗下来，送的2w云豆和实名认证的2000云豆一下子就没了。。然后全部应用的数据全被清空，倒是没什么损失：博客基本没什么文章，其他应用也都是测试性的。

前几天花了10块前充了10软妹币，现在没有租金，我只搭一个应用，应该可以撑一段时间的吧。

由于刚学不久，简单的Demo部署都遇到一些问题。。。譬如需要在应用信息里面设置CallBack页面，说白了很简单：你总得提供一个oAuth认证完毕后跳转的页面吧。

PythonSDK只是简单的尝试，我觉得更方便的是JavaScript的SDK。

JavaScript SDK

JavaScript SDK用起来很爽，首先只要跟着JS SDK 文档一步步下来就行了。

注意以下几点：

  • 微博的API调用需要用户认证，所以在每次调用前都得通过WB2.CheckLogin()检查是否登录。

  • 调用次数有限制！所以获取一条微博的转发列表时要注意！特别是有上万次转发的，每次只能200条，10000次就是50次调用，每小时的配额一下子就用完了=。=我现在的状态就是用完了配额，403 Forbidden，然后来写博客。。。

花了一天两夜的时间，总算把那个小页面做出来了，见http://xinlinqi.oicp.net/wei/，还是截个图吧，原因有两个：

  • 由于是花生壳的动态域名，指不定什么时间就访问不了了。。

  • 接入的网站没有通过审核，只能指定几个可以登录的微博帐号。。。

后记

做自己感兴趣的事情真好，废寝忘食什么的。哈哈~
